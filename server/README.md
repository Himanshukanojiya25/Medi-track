# MediTrack Backend

This folder contains the backend service for the MediTrack Hospital SaaS platform.

## Tech Stack
- Node.js
- Express
- TypeScript
- MongoDB (Mongoose)
- JWT Authentication

## Structure
- `src/` → application source code
- `logs/` → runtime logs
- `uploads/` → uploaded files
- `tests/` → backend tests

## Development
```bash
npm install
npm run dev
